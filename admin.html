Importer plusieurs clients

Collez votre liste ci-dessous. Un client par ligne, format : RAISON SOCIALE ; COMPANY_ID
Séparateur accepté : ;, |, tabulation ou ,
Exemple :
EXEMPLE SAS ; 11111111
EXEMPLE SARL ; 22222222
SCI EXEMPLE ; 33333333
# Les lignes commençant par # sont ignorées
💡 Enrichissement automatique : pour chaque ligne avec un company_id, le backend interroge Pennylane et récupère automatiquement le nom officiel et le SIREN. Si vous laissez le nom vide, il sera rempli automatiquement.
